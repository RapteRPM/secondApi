# Frontend - Sistema de Autenticación

Interfaz web simple para login y registro de usuarios.

## Estructura

- `index.html` - Página principal con formularios de ingreso y registro
- `dashb.html` - Dashboard (panel de control)
- `style.css` - Estilos personalizados

## Tecnologías

- HTML5
- CSS3
- Bootstrap 5.3.0
- JavaScript 

## Ejecución

Abre `index.html` en el navegador o utiliza un servidor local:

```bash
python3 -m http.server 8000
```

Luego accede a: `http://localhost:8000`

## Características

- Formulario de login
- Formulario de registro
- Alternancia entre formularios sin recargar página
- Diseño responsive con Bootstrap
